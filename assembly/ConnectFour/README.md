# Connect Four in MIPS Assembly

A recreation of the classic **Connect Four** game written entirely in **MIPS Assembly** and designed to run on the **MARS (MIPS Assembler and Runtime Simulator)**.

The project demonstrates how a complete terminal-based game can be implemented using low-level programming concepts, including game state management, user input handling, and basic artificial intelligence, all rendered through ASCII graphics.

## Features

- Classic Connect Four gameplay
- Two-player local mode
- Single-player mode against a simple AI opponent
- ASCII-based game interface
- Developed entirely in MIPS Assembly

## Running the project

1. Download and open the project using **MARS (MIPS Assembler and Runtime Simulator)**.
2. Load the Assembly source file.
3. Assemble and run the program.
4. Choose whether to play against another player or the computer.

## Project Goals

- MIPS Assembly programming
- Memory and register management
- Control flow and procedures
- Game logic implementation
- Terminal-based user interfaces
